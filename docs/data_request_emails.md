# CSFS Data Request Templates

Draft emails for requesting streamflow data from agencies without public APIs.
These should be sent from an institutional email address and customized per request.

---

## Latin America & Caribbean

### Costa Rica — Dirección de Agua / ICE
**To:** contacto@da.go.cr / ICE Departamento de Hidrología
**Subject:** Request for historical streamflow data — academic research

Dear Dirección de Agua / ICE,

We are developing an open-source global streamflow observation service (CSFS)
for the hydrology research community. We kindly request access to historical
daily streamflow (caudal) records from your gauging station network in digital
format (CSV or Excel).

Could you advise whether bulk station data is available via SINIGIRH
(mapas.da.go.cr) or if a formal data request is required? We are happy to
comply with any data use agreements and will provide proper attribution.
